Projects

  • YourVoice - nodejs, expressjs, mongoDB

    Create and share poll online with your friends. The application have been built using the boilerplate clementinejs. It was my first project using nodejs. The application use Google graph to render the results.

    Social tool, to find where the mate from your communauty are going. The application use Yelp search API to browse through reviewed locations. The user can indicate where he is going, and see how many people are going.

    Small javascript calculator. I had a lot of fun creating the customs functions for the basic operations (add, multiply, divide, substract). This is one the simplest but also one of my favorite project.

  • Pomodoro Clock - jquery, CSS, HTML

    The Pomodoro clock refers to the Italian tool used to control the cooking time. But in this case, it is a personal efficiency tool. It helps to maximize your concentration by scheduling short break time every X minutes.

  • WikiSearch - jquery, CSS, HTML, wikipedia API

    Small project using Wikipedia API and dynamic CSS animations.

  • Weather app - jquery, CSS, HTML

  • Chuck Norris Quote - Jquery, JSON
